# Chinese-Child-Captions
Chinese Children Image Captioning Dataset(中文儿童图像描述数据集)
### 数据集介绍
- 目前已知最大的儿童图像描述数据集
- 1200张图片
- 每张图片对应五个中文描述，每两张图片为一组
- 描述文字600*5=3000
